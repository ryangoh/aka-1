# aka - The Missing Alias Manager

aka generate/edit/destroy/find permanent aliases with a single command. 

#Requirement
Ruby.

#Install
```
git clone https://github.com/ytbryan/aka.git ~/.aka
cd ~/.aka
bundle
./aka install
```

#Usage
```
aka generate hello="echo helloworld"
aka destroy hello
aka edit hello
aka find hello
aka usage
aka help

```
