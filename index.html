aka - The Missing Alias Manager

  1. A delightful way to manage and grow your command shortcuts from the terminal.
  2. Generate permanent alias with a single command.
  3. Save your workflow.
  4. Save your time.

Usage

	
	aka generate hello="echo helloworld" 
	aka destroy hello
	aka edit hello
	aka find hello
	aka usage
	aka

Install

	
	git clone https://github.com/ytbryan/aka.git ~/.aka
	cd ~/.aka
	bundle
	./aka install
