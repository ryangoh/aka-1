aka

The Terminal's Missing Alias Manager

View Source Code Send us Pull Request Submit Issues

Usage

	
	aka generate hello='echo helloworld' #generate a permanent alias
	aka destroy hello #destroy a permanent alias
	aka edit hello #edit a permanent alias
	aka find hello #find an alias
	aka usage #show the usage
	aka #list your aliases


Install

	
	git clone https://github.com/ytbryan/aka.git ~/.aka
	cd ~/.aka
	bundle
	./aka install


Shorter commands? Sure. 

	
	aka g hello='echo helloworld' #generate a permanent alias
	aka d hello #destroy a permanent alias
	aka e hello #edit a permanent alias
	aka f hello #find an alias
	aka u #show the usage
	aka #list your aliases
