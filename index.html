Usage

	
aka generate hello='echo helloworld' 
aka destroy hello  
aka edit hello  
aka find hello  
aka usage  
aka


Install

	
git clone https://github.com/ytbryan/aka.git ~/.aka
cd ~/.aka
bundle
./aka install


Shorter commands? Sure. 

	
aka g hello='echo helloworld' 
aka d hello
aka e hello 
aka f hello 
aka u
